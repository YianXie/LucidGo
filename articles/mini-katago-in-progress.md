# Mini KataGo is Work in Progress!

We are excited to announce that our [Mini-KataGo](https://github.com/YianXie/Mini-KataGo) project is in work in progress.

## What is Mini-KataGo?

Mini-KataGo is my own implementation of KataGo (though a lot simpler). It uses the same concept and idea as KataGo (NN-Guided MCTS), and allows me to send request to it similar to what I did with KataGo.
