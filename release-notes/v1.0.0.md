# LucidGo v1.0.0

LucidGo is a Go game analysis web app. Upload an SGF file, navigate moves on an interactive board, and get AI-powered position analysis backed by the LucidTree engine.

***

## Features

### Game Board

* Interactive Go board powered by `@sabaki/go-board`
* SGF file upload and parsing
* Real-time play
* Move-by-move navigation with full game history
* Support for passing moves
* Multiple simultaneous boards on the demo page

### AI Analysis

* Win rate display from the perspective of each player
* Top move suggestions with visit counts and policy values
* Win rate graph charted across the game
* Policy heatmap overlay on the board
* Per-move analysis results cached during a session

### Analysis Configuration

* Configurable algorithm, rules, and komi
* Neural network and MCTS parameter tuning (simulations, c\_puct, Dirichlet noise)
* Output options: top move count, win rate, visit counts, policy inclusion
* Save and reuse past analysis configurations
* Per-board settings editor at `/settings/:id/`

### Authentication

* Email-based registration and login (no username required)
* JWT authentication with auto-refresh
* User profile page for account management

### UI & Accessibility

* Sidebar configuration panel
* Tooltip descriptions for all analysis setting fields
* Responsive layout for mobile and desktop

***

## Tech Stack

| Layer     | Technology                                      |
|-----------|-------------------------------------------------|
| Backend   | Python 3.12, Django 5.2, Django REST Framework  |
| Frontend  | React 18, TypeScript, Vite 7, MUI v7, Tailwind v4 |
| Analysis  | LucidTree API (proxied via Django)              |
| Auth      | JWT via `djangorestframework-simplejwt`         |
| SGF       | `sgfmill`                                       |
