# LucidGo v1.1.0

Changes since commit `33ab20a` (improved mobile responsiveness).

## New Features

### Analyze Page

A full-featured `/analyze/` page has been added as the primary analysis experience. It supports SGF upload and live game entry, move-by-move navigation, per-move analysis requests to LucidTree, win rate graphing, and saving/loading past analysis sessions.

### Demo Page for New Users

A dedicated `/demo/` page now provides a pre-loaded sample game so new users can explore the board and analysis features without uploading their own SGF file.

### Navigation Sidebar

A new responsive navigation sidebar (`NavSidebar.tsx`) replaces the previous header-based navigation, providing icon-based links to all main pages with collapsible drawer support.

### Auto-Save Games Setting

A new `auto_save_games` user preference has been added. When enabled, games are automatically saved to the user's account after upload. This preference is stored in a new `general_settings` JSON field on the `UserSettings` backend model.

### Board Interaction Controls

* Move navigation now supports **mouse wheel** and **click**
* Wheel-based navigation is gated behind holding `Cmd` (macOS) or `Ctrl` (Windows/Linux) to avoid accidental scrolling.

### Analysis Config Improvements

* Renamed several config parameters for clarity.
* Refactored analysis function code for maintainability.

### Win Rate Graph

The win rate graph has been visually updated to improve readability.

### Documentation Renderer

The docs page now supports **code syntax highlighting** via an updated MDX renderer.

### Feedback Form Link

A link to the feedback form has been added to the account menu in the header.

## Bug Fixes

* Fixed an incorrect logo displayed on the feedback form.
* Fixed saved games not loading correctly after the `auto_save_games` attribute was introduced.

## UI / Styling

* Updated action bar styling on the Analyze and Demo pages.
* General page styling improvements for better UX across the app.

## CI / Maintenance

* Updated `pip-audit` ignore list for known upstream vulnerabilities.
* Updated `pygments` package.
* Updated `Makefile` targets for local development workflow.
